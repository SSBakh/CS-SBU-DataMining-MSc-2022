this document is about apartment rental offers in germany project , I used the dataset to explore, visual , predic the price of houses based on different features.

2.2. Methods

2.2.1. missing data

  • there are much amount of data missing in this dataset so after checking columns , the ones with more 50% missing value droped , and there was some another columns that was not used because the data was not usefull
  • in categorical columns , based on % of state in category data , chanded to other category
  • the unsual data like livingpace== 0 was removed from data.
  • missing data in categorical column was replace with other state.
  • duplicate data removed form dataset
